#  Real-Time Currency Converter (React)

A simple real-time currency converter built with React and the Frankfurter API.

## Features
- Fetches live exchange rates from a public API.
- Dropdowns for selecting source and target currencies.
- Input field for amount conversion.
- Real-time display of conversion results.
- Handles API errors gracefully.
- Optimized using `useMemo` and `React.memo`.
